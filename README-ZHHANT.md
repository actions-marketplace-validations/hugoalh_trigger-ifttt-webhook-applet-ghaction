#### `webhook_eventname`

**ğŸ…¿** `<string>` IFTTT webhook event name; Recommended to keep in lower case to prevent issue.

**ï¼»é¸æ“‡æ€§ï¼½** `<boolean = false>` Trigger the webhook applet with a standard payload (IFTTT default ingredient namespace `value1`, `value2`, and `value3`) or a custom payload.

- `payload`

**ï¼»é¸æ“‡æ€§ï¼½** `<(boolean | null | string) = false>` Replace undefined placeholder when placeholder is not in the list.

- **`"\\true"`:** Replace the placeholder with `"true"`.
